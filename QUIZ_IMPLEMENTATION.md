## 🚀 Как запустить и тестировать:

### 1. Убедитесь что backend запущен:
\`\`\`bash
cd backend
python manage.py seed_mathway
python manage.py runserver 8080
\`\`\`

### 2. Запустите frontend (в другом терминале):
\`\`\`bash
cd frontend
npm install  # если ещё не установлены зависимости
npm start
\`\`\`

### 3. Откройте браузер:
- Перейдите на http://localhost:4200

### 4. Процесс тестирования:

#### Шаг 1: Регистрация / Логин
- Создайте аккаунт или залогиньтесь
- Нажмите "Login / Register"

#### Шаг 2: Просмотр видео
- Нажмите "Videos" в навигации
- Выберите видео из списка
- Нажмите на видео для открытия детальной страницы

#### Шаг 3: Начните Quiz
- На странице видео нажмите кнопку "📝 Take Quiz"
- Вы увидите quiz с вопросами в русском языке

#### Шаг 4: Ответьте на вопросы
- Выбирайте ответы для каждого вопроса
- Используйте кнопки "Назад" и "Далее" для навигации
- Progress bar показывает сколько вопросов вы уже ответили

#### Шаг 5: Отправьте quiz
- После ответа на все вопросы нажмите "Отправить ответы"
- Вы увидите результаты с баллами

#### Шаг 6: Проверьте Leaderboard
- Нажмите "Посмотреть Leaderboard" или используйте навигацию
- Вы должны увидеться в leaderboard с вашими баллами

## 📝 Особенности реализации:

### Quiz Component Features:
1. **Пошаговая навигация**: Легко перемещаться между вопросами
2. **Валидация**: Проверка что все вопросы ответили перед отправкой
3. **Прогресс**: Visual progress bar показывает выполнение
4. **Результаты**: Показываются баллы, процент и пополнение общего счета
5. **Links**: Быстрые ссылки на видео и leaderboard

### Leaderboard Component Features:
1. **Рейтинг**: Пользователи отсортированы по баллам
2. **Медали**: Топ-3 получают мед 🥇🥈🥉
3. **Автообновление**: Leaderboard обновляется каждые 30 секунд
4. **Дизайн**: Красивое оформление с цветной выделкой топ-3
5. **Responsive**: Хорошо выглядит на мобильных устройствах

## 🔄 Data Flow:

1. User выполняет Quiz
2. Frontend отправляет ответы на Backend
3. Backend:
   - Проверяет правильность ответов
   - Считает баллы
   - Сохраняет QuizAttempt
   - Обновляет Profile.points
4. Frontend получает результат
5. User может посмотреть Leaderboard
6. Leaderboard показывает updated баллы

## 🐛 Troubleshooting:

### Quiz не загружается
- Проверьте что backend запущен
- Убедитесь что video_id правильный
- Проверьте консоль браузера для ошибок

### Leaderboard пуст
- Убедитесь что пользователи прошли quiz
- Точки должны были добавиться автоматически
- Нажмите "🔄 Обновить" для ручного обновления

### Баллы не отображаются
- Проверьте что backend правильно считает баллы
- Убедитесь что quiz имеет вопросы с правильными ответами
